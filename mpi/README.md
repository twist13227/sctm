# Воспроизведение результатов MPI

## Сборка

```bash
module load SpectrumMPI
mpic++ -std=c++11 mpi_solve.cpp -O2 -o mpi_solve
```

## Запуск экспериментов

```bash
chmod +x mpi_submit.sh
./mpi_submit.sh
```

## Результаты

Результаты сохраняются в виде файлов вида:

```
try_mpi_<procs>_1_<N>_<L>_<run_num>.out
```

где:
- `<procs>` — число MPI-процессов (1, 2, 4, 8, 16, 32),
- `<N>` — размер задачи (128 или 256),
- `<L>` — значение параметра L (1 или Pi),
- `<run_num>` — номер запуска.
Все результаты сохраняются в той же директории, откуда запускается `mpi_submit.sh`.
Пример полученных результатов приведён в `mpi_jobs_output.zip`.
