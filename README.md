# Решение заданий по курсу «Суперкомпьютерное моделирование и технологии», читаемому в 3 семестре магистратуры ВМК МГУ

## Структура репозитория

Репозиторий содержит 5 папок под различные <version> задания: последовательной, OpenMP, MPI, MPI+OpenMP и MPI+CUDA.
В каждой из них находятся:
- README.md,
- отчёт `<version>_report.pdf`,
- код и его бинарь `<version>_solve.<extension>` и `<version>_solve`,
- скрипт для запуска `<version>_submit.sh`,
- архив с LSF-выводами, используемыми для отчёта `<version>_jobs_output.zip`,
- дополнительные файлы (опционально).
