# Воспроизведение результатов OpenMP

## Сборка

```bash
g++ -fopenmp -std=c++11 omp_solve.cpp -O2 -o omp_solve
```

## Запуск экспериментов

Для запуска экспериментов нужна папка `submit_jobs`, содержащая lsf-файлы.
Эта папка имеется в репозитории, её нужно распаковать.

```bash
unzip submit_jobs.zip
chmod +x omp_submit.sh
./omp_submit.sh
```

## Результаты

Результаты сохраняются в виде файлов вида:

```
out_<run_num>_<N>_1_<threads>.out
```

где:
- `<run_num>` — номер запуска,
- `<N>` — размер задачи (128 или 256),
- `<threads>` — количество OpenMP-потоков (1, 2, 4, 8, 16, 32).

Все результаты сохраняются в той же директории, откуда запускается `omp_submit.sh`.
Пример полученных результатов приведён в `omp_jobs_output.zip`.
